# Generic anchor lexicon — phrases that operators routinely flag as
# "click here" / SEO-template noise. Used by
# ``apps/pipeline/services/anchor_garbage_signals.py`` to penalise
# anchors that match any line. One phrase per line; lower-case;
# blank lines + ``#`` comments ignored.
#
# Sources:
# - Nielsen Norman Group (2012). "Beyond Blue Links: Making Clickable
#   Elements Recognizable" — link-text quality guidance.
# - W3C WCAG 2.1 §2.4.4 "Link Purpose (In Context)" — accessibility
#   guidance against generic link text.
# - 2024 Google Quality Rater Guidelines §5.3 — explicit
#   "click here" / "read more" examples cited as low-quality anchors.
#
# Curated list of ~120 phrases. Operators add more via
# ``AppSetting["generic_anchor_matcher.extra_phrases"]`` (one per
# line, same format).
